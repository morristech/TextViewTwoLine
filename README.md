# TextViewTwoLine

Add to a module in project

```gradle
allprojects {
   repositories {
      ...
      maven { url 'https://jitpack.io' }
   }
}
   
dependencies {
   compile 'com.github.kimcy929:TextViewTwoLine:1.0.1'
}
```
